#### Mailcow Certificate Automation (2026-01-24)
**Challenge**: Mailcow ACME only supports HTTP-01, but Lady has no port 80 (Traefik uses it)

**qui3tly's Requirement**: "Fix it right. No patches. Must survive reboot."

**Solution Architecture**:
1. **Traefik** (Cloudflare DNS-01) ‚Üí Gets Let's Encrypt cert for mail.quietly.online

**Implementation**:
- 3 Python scripts: cert extract, TLSA update, MTA-STS TXT update
- systemd service + timer: ExecStartPre ‚Üí ExecStart ‚Üí ExecStartPost
- Tested: Cert renewal triggers full chain ‚úÖ

**Result**: Zero-touch certificate renewal with DANE/TLSA security

#### Headscale DNS Bootstrap - Critical Fix (2026-01-17)
**Incident**: 363 Headscale crash restarts, 2 hours downtime

**Root Cause Analysis** (qui3tly):
- Tailscale overwrote /etc/resolv.conf ‚Üí MagicDNS (100.100.100.100)
- Headscale needs real DNS to fetch DERPMap on boot
- MagicDNS needs Headscale running
- **Circular dependency = crash loop**

**Solution**:
1. Master /etc/resolv.conf ‚Üí `nameserver 1.1.1.1` (Cloudflare)
2. File immutable: `chattr +i /etc/resolv.conf`
3. Boot order enforced: Headscale ‚Üí Tailscale (systemd After=)
4. Documentation: HEADSCALE_OPERATIONS.md updated with detailed explanation

**Result**: Zero Headscale crashes since fix. Rock-solid stability.

**Lesson**: qui3tly spotted the circular dependency instantly. 30 years of experience knows what breaks.
